Limited support for declarative agents is available for [Microsoft 365 Government](https://www.microsoft.com/microsoft-365/government) tenants:

- **Government Community Cloud (GCC)**: Full declarative agent support is available.
- **Government Community Cloud High (GCCH)**: Limited support — only pro-code agents built with Microsoft 365 Agents Toolkit are supported. Agent Builder isn't available for declarative agents in GCCH.
- **Department of Defense (DoD)**: Declarative agents aren't currently available.
